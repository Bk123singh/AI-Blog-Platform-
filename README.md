# 🤖 AI Blog Platform

🚀 A full-stack MERN-based blog platform that leverages AI APIs for dynamic content generation, featuring secure authentication & authorization,  error handling, and a modern responsive UI.

---

## 📌 Features

* 🤖 AI-powered blog content generation
* 🔐 Authentication & Authorization (JWT-based)
* 🛠️ RESTful API architecture
* ⚠️ Error handling and validation
* 📂 Category-based blog filtering
* 🧑‍💻 Admin Dashboard for managing blogs, users, and content
* 🎨 Responsive UI with smooth animations (React + Tailwind CSS)

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Tailwind CSS
* Framer Motion

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

**Other Tools & Features:**

* JWT Authentication
* AI API Integration
* REST API



---

## 🚀 Getting Started

### 2. Install dependencies

```bash
npm install
```

### 3. Setup environment variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
AI_API_KEY=your_ai_api_key
```

### 4. Run the project

```bash
npm run dev
```
